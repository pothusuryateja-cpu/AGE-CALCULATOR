 Age Calculator — Python

A simple Python program that calculates a person’s age using their date of birth.
The program uses Python’s `datetime` module to accurately compute the age in years (and optionally months and days).

---
 Features

* Calculates age based on Date of Birth
* Handles real calendar dates accurately
* Beginner-friendly code
* Works in any terminal or IDE
* Easy to extend or modify

---
 Tech Stack

* Python 3
* `datetime` module

---
 Installation and Usage

Make sure Python is installed, then run the script:

```bash
python age_calculator.py
```

Enter your date of birth in the required format when prompted.

Example:

```
Enter your date of birth (DD-MM-YYYY): 25-12-2000
You are 24 years old.
```
